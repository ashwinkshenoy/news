#BuyBlue Landing Page

###Installation

Initial yeoman setup
`npm install --global yo gulp-cli bower generator-webapp`

Do `npm install`

If bower components do not get installed,
do `bower install`

###To run the project

`gulp serve`

###To build the project

`gulp`

###To run built project

`gulp serve:dist`
